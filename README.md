# Tkinter-Calculator
# 🧮 Tkinter Calculator

A simple **GUI Calculator application** built using **Python and Tkinter**.  
This is my first Python project, created to practice GUI development, functions, and event handling.

## 📌 About the Project

The Tkinter Calculator is a desktop calculator that allows users to perform basic arithmetic operations through an easy-to-use graphical interface.

This project helped me understand:
- Python GUI development using Tkinter
- Creating buttons and layouts
- Handling user inputs
- Connecting functions with GUI events

## ✨ Features

- Addition
- Subtraction
- Multiplication
- Division
- root
- log
- ln
- square
- 🔢 User-friendly button interface
- 🖥️ Desktop GUI application
- ⚡ Instant calculation results

## 🛠️ Technologies Used

- **Python**
- **Tkinter** (Python GUI Library)

## 🚀 How to Run

1. Clone this repository:

```bash
git clone https://github.com/Piyush-Patil1306/Tkinter-Calculator.git
```

2. Go into the project folder:

```bash
cd Tkinter-Calculator
```

3. Run the Python file:

```bash
python calculator.py
```
